# .debug file

example

```markup
<?xml version="1.0" encoding="UTF-8"?>
<ExtensionList>
    <Extension Id="com.autoEdit.it">
        <HostList>
            <Host Name="PPRO" Port="8099"/>
        </HostList>
    </Extension>
</ExtensionList>
```

Then you can open chrome at [http://localhost:8099/](http://localhost:8099/) to view the console for the panel
